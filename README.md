Cherryline, a status line
-------------------------

Depends on [tpope/vim-fugitive](https://github.com/tpope/vim-fugitive) for detecting the git branch.

Configuration variables:
* `g:cherryline_no_repo` (default `OwO`): String to display instead of a git branch, when not in a git repository
* `g:cherryline_no_fugitive` (default `’’`): String to display instead of a git branch, when fugitive is not installed
